<h1 align="center"><span style="color:#dc2626;">Barik Wrapped</span></h1>

Barik Wrapped es un visor de estadísticas personales de la tarjeta Barik.  Disponible en **[barikwrapped.danialonso.dev](https://barikwrapped.danialonso.dev)**.

¿Quieres mejorar algo? Las PRs y contribuciones son bienvenidas.

## Cómo ejecutar el proyecto

1. **Instala dependencias**
	```bash
	npm install
	```
2. **Arranca el entorno de desarrollo**
	```bash
	npm run dev
	```
3. Abre `http://localhost:3000` en tu navegador.

Para generar el build de producción:

```bash
npm run build
npm start
```
